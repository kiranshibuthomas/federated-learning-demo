How Federated Learning Works

  1. Local Training: Each user's model trains on their own data locally
  2. Model Updates: Only model weights (not data) are shared
  3. Federated Averaging: Server aggregates weights to create global model
  4. Distribution: Updated global model sent back to all users
  5. Privacy: Raw data never leaves the user's device
