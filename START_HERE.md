## 🎯 What This Project Does

### The Problem
Traditional machine learning requires sending your data to a server, which creates privacy risks. Your keyboard sees everything you type - passwords, messages, personal information.

### The Solution
**Federated Learning**: Train models locally on your device, only share model updates (not data) with the server. This is how Google Gboard works with 1+ billion users!

### The Demo
Three users with different interests:
- ⚽ **Alex** (Sports): Gets predictions like "game", "match", "player"
- 🧁 **Bailey** (Baking): Gets predictions like "flour", "oven", "recipe"
- 💻 **Chris** (CS): Gets predictions like "code", "function", "algorithm"

Each user's data stays private, but all benefit from federated learning!
